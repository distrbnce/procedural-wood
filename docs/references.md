# References

This project builds on public, long-standing procedural graphics techniques:

- Ken Perlin, "An Image Synthesizer" (1985), for gradient noise.
- Ebert et al., "Texturing and Modeling: A Procedural Approach", for classic procedural texture patterns.
- Public shader community examples using rings, turbulence, domain warping, and fBm.
- Wood anatomy concepts such as softwood, ring-porous hardwood, diffuse-porous hardwood, pores, rays, and growth rings.

The project goal is not to claim a single canonical formula for each species. It is to document reproducible algorithm profiles that approximate recognizable wood traits.
