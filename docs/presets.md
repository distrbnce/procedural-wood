# Presets

Each preset uses the same parameter vocabulary so implementations can share controls.

## Shared Parameters

```json
{
  "ringFrequency": 8.0,
  "ringContrast": 0.5,
  "warpStrength": 0.4,
  "warpScale": 1.5,
  "grainStrength": 0.5,
  "grainStretch": 8.0,
  "knotDensity": 0.2,
  "poreDensity": 0.3,
  "rayIntensity": 0.2,
  "baseColor": "#a66a3a",
  "latewoodColor": "#4f2b17"
}
```

## Tuning Notes

- Pine should prioritize broad bands and knots.
- Oak should prioritize pore bands and rays.
- Walnut should prioritize dark smooth variation and flowing warp.
- Maple and birch should keep contrast low.
- Cherry should use warm red-brown color drift with fine grain.
- Mahogany should support ribbon stripe figure.
- Teak should use golden tones with dark oily streaks.
