# Playground

Open `index.html` in a browser to render Earth/Sun Spirograph: Procedural Woodgrain Designer.

The name frames woodgrain as an Earth/Sun drawing system: a procedural echo of ultraviolet light, seasonal growth, and Earth's changing tilt in relation to the Sun.

Features:

- Select a species preset.
- Tweak ring, warp, grain, pore, ray, knot, ribbon, and oil-streak parameters.
- Compare side grain and end grain.
- Export the current preset as JSON.
